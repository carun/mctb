புத்தரின் ஆழ்ந்த போதனைகளைச் சிறப்புற கற்றல்
----------------------------------------------------

இது `Dr. டேனியல் இங்ரம் <https://www.integrateddaniel.info/>`_ "Mastering the Core Teachings of the Buddha"
என்ற பெயரில் எழுதி 2018ஆம் வெளியிட்ட ஆங்கில நூலின் தமிழாக்கம்.

`மூல ஆங்கில நூலைப் படிக்க விருப்பமுள்ளவர்கள் இங்கு சொடுக்கவும் <https://www.mctb.org/mctb2>`_

- இந்நூலின் பதிப்புரிமை, காப்புரிமை மற்றும் அனைத்து உரிமமைகளும் `Dr. டேனியல் இங்ரமைச் <https://www.integrateddaniel.info/>`_ சாரும்
- அவரை `ட்விட்டரில் பின்தொடர <https://twitter.com/danielmingram?ref_src=twsrc%5Etfw>`_

.. note::

   வேலை நடந்து கொண்டிருக்கிறது. இது வரைவு மட்டுமே.
